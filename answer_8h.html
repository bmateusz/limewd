Answer to connection callback function.

It can be called multiple times during a single request.

  • During the first call only the headers are present.
  • For the second call we can create the response.
  • If it was a POST, it can be called several times with upload_data.
Parameters:
clscustom value selected at registration time
connectionthe connection identifying the client
urlthe URI called from the client
methodrequest type, practically POST or GET
versionHTTP version
upload_datafilled in case of POST with upload data
upload_datafilled in case of POST with upload data size
ptrcustom parameter, contains a Request struct after the first call
Returns:
MHD_YES on success, MHD_NO on error which causes the connection to terminate
